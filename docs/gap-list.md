# 공식 라이브러리 갭 리스트 (§21 확장 큐)

작성 2026-07-04. **검증 기준: KiCad 공식 GitLab master** (kicad.github.io 스냅샷 아님) + 오픈 MR 조회.
방법: 심볼은 `*.kicad_symdir` 전수 트리 리스팅(페이지네이션), 풋프린트/3D는 디렉터리 전수, MR은 키워드 검색. 2개 독립 에이전트 교차검증.

## 방법론 교훈 (다음 채굴 때 반드시)

1. **kicad.github.io는 릴리즈 스냅샷** — master와 다름. 1차 조사 톱10 중 8개가 이것 때문에 오탐 (ESP32-S3, TP4056, CH224K, RP2040, JST ZH, USB-C 6P, INA219, JST SH 3D 전부 master에 존재).
2. **비직관적 라이브러리 배치가 오탐 유발**: INA219→Sensor_Energy, nRF24L01 심볼→RF(풋프린트는 RF_Module), DHT22→AM2302 이름, TP4056→패키지 접미사(TP4056-42-ESOP8). 한 라이브러리만 이름 검색하면 거짓 부재 판정.
3. **오픈 MR 확인 필수**: MR 진행 중인 갭은 수명이 짧음(만들어도 곧 공식이 커버). 단 2022년부터 방치된 MR은 공식이 안 움직이는 영역이라는 신호 = 오히려 기회.
4. 착수 직전 **master 재확인** (이 리스트도 시간이 지나면 낡음).
